# 啟動中繼廣告

Adobe GenStudio for Performance Marketing支援為Instagram和Facebook啟用中繼廣告或創意。

您可以在GenStudio for Performance Marketing中[建立中繼體驗](/help/user-guide/create/create-meta-ad.md)，並選取它以啟動，或從[!DNL Activate]中已核准的資產建構新的體驗。

啟用中繼廣告會遵循啟用其他付費管道所需的[相同的一般步驟](create-activation.md)。 啟用程式支援針對中繼的特定需求準備您的廣告體驗。 在GenStudio for Performance Marketing中啟動中繼體驗（或創意）後，使用[中繼廣告管理員](https://adsmanager.facebook.com/)在最終發佈之前微調特定中繼廣告位置的體驗。

## 第 1 步：設定您的 Meta 帳戶

在啟動啟用之前，請 [先登錄元維琪](https://adsmanager.facebook.com/) 以訪問您的元廣告管理工具帳戶。 確認您已關聯的 Meta 廣告 帳戶包含：

* Facebook 頁面
* 元行銷活動
* 元廣告集
* Instagram 設定檔 （選用）

您必須擁有權限才能貼文內容元廣告管理工具。

## 第 2 步：關聯至您的 Meta 帳號

在您的組織激活體驗之前，GenStudio 系統管理員需要將您的 Meta 帳戶連接到 GenStudio 以進行績效行銷。 此連接允許數據在 GenStudio 和 Meta 按讚外部行銷工具之間流動，從而實現啟用流程。

請参閱 [連接到元 （Facebook） 帳戶](/help/user-guide/connectors/connect-channel.md#meta-ads-connect)。

同步完成後，您可以視圖新增的帳戶。 大量數據需要較長的時間才能同步。

## 第 3 步：為啟用準備體驗

GenStudio 系統管理員和编辑可以啟動廣告體驗。

**若要準備您的啟動體驗**：

1. 從&#x200B;_[!DNL Activate]_，按一下中繼產品圖磚上的&#x200B;**[!UICONTROL 新增]**。_ Creative安裝程式&#x200B;_檢視隨即開啟。

   Creative設定頁面提供了一個中心位置，供您準備 Meta 廣告 啟用。 準備廣告包括以下三項任務：

1. 為您的體驗指定一個名稱。 啟用后，您可以使用此名稱在「已啟動的體驗&#x200B;_」表格中_&#x200B;搜尋此體驗。
1. 选择媒體資產。 你可以使用來自内容或上傳外部資產（例如，從 OneDrive 或 Dropbox）資產。
1. [添加文字](#add-ad-text)。
1. [新增中繼資料](#assign-metadata)。

   _預覽面板_&#x200B;支援文字和資產在特定廣告投放內容中的互動式檢視。 使用&#x200B;_選取位置_&#x200B;下拉式功能表，在支援的廣告位置之間切換。 預覽提供完成特定位置廣告元素決策的機會。 當您在&#x200B;_預覽_&#x200B;面板中選取版位時，只有您的廣告檢視會受到影響。 未儲存您在&#x200B;_預覽_&#x200B;面板中的位置選取範圍。

### 選取您的媒體資產

使用&#x200B;_媒體_&#x200B;區段來選取至少一個影像資產以包含在您的體驗中。 廣告版位與支援的影像外觀比例相關聯，這些比例在&#x200B;_版位_&#x200B;下拉式功能表中列為選項。 此功能表會依外觀比例顯示Facebook貼文或Instagram內文支援的廣告投放位置。

上傳後，資產會儲存至&#x200B;_[!DNL Content]_。_&#x200B;媒體&#x200B;_區域預設會以1:1的外觀比例顯示影像。 替代外觀比例僅包含付費廣告頻道支援的值。 它們會依垂直和水平方向分組。 GenStudio for Performance Marketing支援每個啟用的體驗最多包含六個外觀比例。

**若要從內容上傳資產**：

_[!DNL Content]_提供組織已核准資產和體驗的中央檢視。 您可以使用Search（放大鏡）和篩選功能表選項聚焦[_[!DNL Content]_ **** 庫顯示的資產](/help/user-guide/content/manage-assets.md)清單。__

1. 在 中 _[!DNL Activate]_，按兩下&#x200B;**[!UICONTROL 元廣告磁貼上的新]**。 Creative_&#x200B;設定&#x200B;_視圖隨即開啟。

1. 按兩下 **[!UICONTROL 選擇]**，然後選擇 **[!UICONTROL 從內容]**&#x200B;中選擇。 此時將打開“ _選擇內容”視圖_ ，其中显示搜尋或篩選的圖像資產庫。

1. 使用圖庫的搜尋和篩選器工具至少 _[!DNL Content]_選擇一個上傳資產。

1. 按一下&#x200B;**[!UICONTROL 使用]**&#x200B;將選取的資產包含在您的創意中。 _Creative設定_&#x200B;視窗將資產納入其預設外觀比例中，位於&#x200B;_媒體_&#x200B;區域。 _預覽_&#x200B;面板在支援此外觀比例的廣告投放中預覽資產。

如果上傳失敗，會開啟資訊性錯誤訊息，其中包含&#x200B;_[!DNL Content]_中資產的連結。

**若要上傳外部資產**：

您可以從Microsoft OneDrive或Dropbox上傳最多6個外部至&#x200B;_[!DNL Content]_相簿的靜態影像。

1. 從&#x200B;_[!DNL Activate]_，按一下中繼圖磚上的&#x200B;**[!UICONTROL 新增]**。_ Creative安裝程式&#x200B;_視窗隨即開啟。

1. 在&#x200B;_媒體_&#x200B;區段中，按一下&#x200B;**[!UICONTROL 選取]**。 下拉式功能表會顯示&#x200B;_從「內容」_&#x200B;或&#x200B;_上傳_&#x200B;中選取的選項。

1. 按一下 **[!UICONTROL 上傳]**。 _加入外觀比例_&#x200B;視窗開啟。

1. 將影像檔案拖放至影像上傳區域，選取支援外觀比例的影像。 或者，您也可以瀏覽裝置以尋找資產。

1. （選擇性）若要從您的裝置上傳資產，請按一下[瀏覽] ****，然後選取[瀏覽]檔案&#x200B;_或[瀏覽]資料夾_&#x200B;以識別要上傳的資產。__

1. 在&#x200B;_新增詳細資料_&#x200B;區域，新增資訊性詳細資料至您上傳的資產，以便在&#x200B;_[!DNL Content]_中搜尋和篩選。 這些詳細資料會儲存為中繼資料。

1. 上傳資產和指派的詳細資料後，請按一下右下方的&#x200B;**[!UICONTROL 新增Assets]**。

### 新增廣告文字

__&#x200B;使用激活元廣告&#x200B;_頁面的文本_&#x200B;部分，將引人注目的品牌文本添加到必填文本字段。文字包含廣告的主要（內文）文字和行動號召文字。

| 欄位 | 必填 | 字元限制（上限） |
|-----------------|---------------------------|---------------------------------|
| 廣告名稱 | 是 | 500 |
| 原文 | 是的 | 500 |
| 標題 | 是的 | 255 |
| 說明 | 不 | 125 |
| 呼叫動作 | 是的 | 僅限下拉功能表選項 |
| 顯示URL | 否 | 1000 |
| 網站URL | 是的 | 1000 |
| 影像 | 至少需要一個 |                                 |

_GenStudio 僅針對效果行銷需要主要文本_ 和 _標題_ ，而不是元維琪。

### 指派中繼資料

體驗詳細資料會儲存為中繼資料，並可在搜尋體驗時協助使用者。 這些詳細資料可提升[!DNL Content]中體驗的可見度。 使用這些選擇性的使用者定義詳細資訊，來識別體驗的用途以及部署體驗的內容或行銷活動。

| 詳細資料 | 說明 |
|------------|-------------|
| 行銷活動 | 廣告 體驗 所屬的所有 GenStudio for Performance Marketing 營銷活動 |
| 品牌 | 用戶定義的或預設的準則，允許使用者建立品牌準則，以捕捉品牌身份的本質。 |
| 產品 | 與您的組織相關聯並在 GenStudio for Performance Marketing 中標識的產品 |
| 角色 | 與您的組織相關聯並在 GenStudio 中標識的角色，用於績效行銷 |
| 時間範圍 | 廣告體驗處於活動狀態的季度、季節、年或其他組織定義的時間單位 |
| 地區 | 啟動體驗的地理區域 |
| 語言 | 使用廣告體驗的語言 |
| 關鍵字 | 便於搜尋和分類廣告體驗的使用者定義關鍵字 |

組合您的創意之後，按一下&#x200B;**[!UICONTROL 下一步]**&#x200B;以確認您的中繼設定。

## 步驟4：確認中繼帳戶設定

準備創意後，您必須確認您的中繼帳戶資訊，並為廣告體驗指派追蹤ID。 _中繼廣告設定_&#x200B;檢視已填入衍生自已設定中繼帳戶的選項。

| 詳細資料 | 說明 |
|------------|-------------|
| 帳戶 | 已連線至GenStudio for Performance Marketing的中繼帳戶 |
| Facebook頁面 | 發佈體驗的Facebook頁面 |
| Instagram帳戶 | 已連接至 GenStudio 以進行績效行銷的 Instagram 帳戶 |
| 行銷活動 | 廣告體驗所屬的中繼行銷活動 |
| 廣告集 | 啟用的廣告體驗所屬的中繼廣告集。 此設定會決定廣告的最終版位。 |

### 追蹤ID

追踪 ID （廣告名稱） 提供了一種機制，用于收集與體驗績效關聯的量度。 在此欄位中輸入廣告名稱。

按下 **[!UICONTROL 右上方的下一個]** ，預覽您的廣告體驗並完成啟用。

## 第 5 步：預覽並激活您的廣告

查看&#x200B;__&#x200B;頁面將显示在Creative設置&#x200B;_中_&#x200B;組合的廣告體驗，並提供視圖和编辑體驗的最終商機。按兩下 **[!UICONTROL Creative 編輯 設定&#x200B;_卷標旁邊的_區域]**&#x200B;以開始變更。您也可以按兩下&#x200B;**[!UICONTROL 右上角的返回]**&#x200B;返回Creative設定&#x200B;__&#x200B;頁面。

### 完整應用程式廣告體驗的啟用

點擊&#x200B;**[!UICONTROL 發佈]**。完整的元廣告體驗及其相關中繼資料將直接推送到选定的元廣告管理工具廣告集中。 傳遞到 Meta Ads Manager 的體驗會停用。 在元廣告管理工具中，您可以管理部署廣告體驗和元行銷活動的最後步驟。

### 登入 Meta 廣告管理工具以完成啟用

啟用完成後，您必須登錄元廣告管理工具。 在 Meta 廣告管理工具](https://adsmanager.facebook.com/)中[，您可以查看廣告體驗並最終確定發佈到特定 Meta 渠道。
