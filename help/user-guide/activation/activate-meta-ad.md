# 啟動中繼廣告

Adobe GenStudio for Performance Marketing支援為Instagram和Facebook啟用中繼廣告或創意。

您可以在GenStudio for Performance Marketing中[建立中繼體驗](/help/user-guide/create/create-meta-ad.md)，並選取它以啟動，或從[!DNL Activate]中已核准的資產建構新的體驗。

啟用中繼廣告會遵循啟用其他付費管道所需的[相同的一般步驟](create-activation.md)。 啟用程式支援針對中繼的特定需求準備您的廣告體驗。 在GenStudio for Performance Marketing中啟動中繼體驗（或創意）後，使用[中繼廣告管理員](https://adsmanager.facebook.com/)在最終發佈之前微調特定中繼廣告位置的體驗。

## 步驟1：設定您的中繼帳戶

在啟動啟動之前，[登入Meta](https://adsmanager.facebook.com/)以存取您的Meta Ads管理員帳戶。 確認您連線的中繼廣告帳戶包括：

* Facebook頁面
* 中繼行銷活動
* 中繼廣告集
* Instagram設定檔（選填）

您必須有許可權才能將內容發佈至中繼廣告管理員。

## 步驟2：連線至您的中繼帳戶

在貴組織啟動體驗之前，GenStudio系統管理員需要將您的中繼帳戶連結到GenStudio for Performance Marketing。 此連線可讓資料在GenStudio和外部行銷工具（例如Meta）之間流動，以啟用啟用程式。

請參閱[連線至中繼(Facebook)帳戶](/help/user-guide/connectors/connect-channel.md#meta-ads-connect)。

同步完成後，您可以視圖新增的帳戶。 大量數據需要較長的時間才能同步。

## 第 3 步：為啟用準備體驗

GenStudio 系統管理員和编辑可以啟動廣告體驗。

**若要準備您的啟動體驗**：

1. 從&#x200B;_[!DNL Activate]_，按一下中繼產品圖磚上的&#x200B;**[!UICONTROL 新增]**。_ Creative安裝程式&#x200B;_檢視隨即開啟。

   Creative設定頁面提供了一個中心位置，供您準備 Meta 廣告 啟用。 準備廣告包括以下三項任務：

1. 為您的體驗指定一個名稱。 啟用后，您可以使用此名稱在「已啟動的體驗&#x200B;_」表格中_&#x200B;搜尋此體驗。
1. 选择媒體資產。 你可以使用來自内容或上傳外部資產（例如，從 OneDrive 或 Dropbox）資產。
1. [添加文字](#add-ad-text)。
1. [新增中繼資料](#assign-metadata)。

   _預覽面板_&#x200B;支持在特定廣告刊登上下文中交互式視圖文本和資產。_使用選取刊登_&#x200B;下拉功能表在支援的廣告版位之間切換。預覽可讓您商機最終決定特定位置的廣告元素。 在 _預覽_ 面板中選取刊登時，僅影響廣告的視圖。 不會儲存您在「預覽」面板中選取 _的刊登_ 。

### 选择您的媒體資產

使用&#x200B;_媒體_&#x200B;區段來選取至少一個影像資產以包含在您的體驗中。 廣告版位與支援的影像外觀比例相關聯，這些比例在&#x200B;_版位_&#x200B;下拉式功能表中列為選項。 此功能表會依外觀比例顯示Facebook貼文或Instagram內文支援的廣告投放位置。

上傳後，資產會儲存至&#x200B;_[!DNL Content]_。_&#x200B;媒體&#x200B;_區域預設會以1:1的外觀比例顯示影像。 替代外觀比例僅包含付費廣告頻道支援的值。 它們會依垂直和水平方向分組。 GenStudio for Performance Marketing支援每個啟用的體驗最多包含六個外觀比例。

**若要從內容上傳資產**：

_[!DNL Content]_提供組織已核准資產和體驗的中央檢視。 您可以使用&#x200B;**[!UICONTROL 搜尋]**（放大鏡）和_&#x200B;篩選&#x200B;_功能表選項，將[_[!DNL Content]_&#x200B;收藏館](/help/user-guide/content/manage-assets.md)顯示的資產詳細目錄設定為焦點。

1. 從&#x200B;_[!DNL Activate]_，按一下「中繼廣告」方塊上的&#x200B;**[!UICONTROL 新增]**。_ Creative安裝程式&#x200B;_檢視隨即開啟。

1. 按一下&#x200B;**[!UICONTROL 選取]**，然後選取&#x200B;**[!UICONTROL 從內容選取]**。 _選取內容_&#x200B;檢視會開啟，顯示您搜尋或篩選的影像資產庫。

1. 使用&#x200B;_[!DNL Content]_相簿的搜尋和篩選工具，選取至少一個要上傳的資產。

1. 按兩下使用&#x200B;]**，將**[!UICONTROL &#x200B;所選資產納入創意內容。“_Creative設置_”窗口在“媒體&#x200B;_”_&#x200B;區域中以預設縱橫比包含資產。_“預覽_”面板預覽支持此縱橫比的廣告刊登中的資產。

如果上傳失敗，則會打開一條信息性錯誤消息，其中包含對 中 _[!DNL Content]_資產連結。

**要上傳外部資產**..

你最多可以從 OneDrive 或 Dropbox Microsoft庫外部上傳 _[!DNL Content]_六個靜態圖像。

1. 從&#x200B;_[!DNL Activate]_，按一下中繼圖磚上的&#x200B;**[!UICONTROL 新增]**。_ Creative安裝程式&#x200B;_視窗隨即開啟。

1. 在&#x200B;_媒體_&#x200B;區段中，按一下&#x200B;**[!UICONTROL 選取]**。 下拉式功能表會顯示&#x200B;_從「內容」_&#x200B;或&#x200B;_上傳_&#x200B;中選取的選項。

1. 按一下 **[!UICONTROL 上傳]**。 _加入外觀比例_&#x200B;視窗開啟。

1. 將影像檔案拖放至影像上傳區域，選取支援外觀比例的影像。 或者，您可以瀏覽裝置以獲取資產。

1. （選擇性）若要從您的裝置上傳資產，請按一下[瀏覽] ****，然後選取[瀏覽]檔案&#x200B;_或[瀏覽]資料夾_&#x200B;以識別要上傳的資產。__

1. 在&#x200B;_新增詳細資料_&#x200B;區域，新增資訊性詳細資料至您上傳的資產，以便在&#x200B;_[!DNL Content]_中搜尋和篩選。 這些詳細資料會儲存為中繼資料。

1. 上傳資產和指派的詳細資料後，請按一下右下方的&#x200B;**[!UICONTROL 新增Assets]**。

### 新增廣告文字

使用&#x200B;_啟動中繼廣告_&#x200B;頁面的&#x200B;_文字_&#x200B;區段，將引人入勝且符合品牌的文字新增至必要的文字欄位。 文字包含廣告的主要（內文）文字和行動號召文字。

| 欄位 | 必填 | 字元限制（上限） |
|-----------------|---------------------------|---------------------------------|
| 廣告名稱 | 是 | 500 |
| 主要文字 | 是 | 500 |
| 標題 | 是 | 255 |
| 說明 | 否 | 125 |
| 行動號召 | 是的 | 僅限下拉功能表選項 |
| 顯示URL | 否 | 1000 |
| 網站URL | 是的 | 1000 |
| 影像 | 至少需要一個 |                                 |

_GenStudio 僅針對效果行銷需要主要文本_ 和 _標題_ ，而不是元維琪。

### 指派中繼資料

體驗詳細数据會儲存為中繼資料，並在搜尋體驗時協助使用者。 這些細節提高了體驗在 中的 [!DNL Content]可見性。 使用這些可選、用戶定義的詳細信息來確定體驗的用途以及部署的上下文或營銷活動。

| 細節 | 說明 |
|------------|-------------|
| 行銷活動 | 廣告 體驗 所屬的所有 GenStudio for Performance Marketing 營銷活動 |
| 品牌 | 用戶定義的或預設的準則，允許使用者建立品牌準則，以捕捉品牌身份的本質。 |
| 產品 | 與您的組織相關聯並在 GenStudio for Performance Marketing 中標識的產品 |
| 角色 | 與您的組織相關聯並在 GenStudio 中標識的角色，用於績效行銷 |
| 時間範圍 | 廣告體驗處於活動狀態的季度、季節、年或其他組織定義的時間單位 |
| 地區 | 啟動體驗的地理區域 |
| 語言 | 使用 廣告 體驗 的語言 |
| 關鍵字 | 使用者定義的關鍵字，便于搜尋和分類廣告體驗 |

組装好創意內容后，按兩下 **[!UICONTROL 一個]** 以確認您的 Meta 設置。

## 步驟4：確認中繼帳戶設定

準備創意後，您必須確認您的中繼帳戶資訊，並為廣告體驗指派追蹤ID。 _中繼廣告設定_&#x200B;檢視已填入衍生自已設定中繼帳戶的選項。

| 詳細資料 | 說明 |
|------------|-------------|
| 帳戶 | 已連線至GenStudio for Performance Marketing的中繼帳戶 |
| Facebook頁面 | 發佈體驗的Facebook頁面 |
| Instagram帳戶 | 已連線至GenStudio for Performance Marketing的Instagram帳戶 |
| 行銷活動 | 廣告體驗所屬的中繼行銷活動 |
| 廣告集 | 啟用的廣告體驗所屬的中繼廣告集。 此設定會決定廣告的最終版位。 |

### 追蹤ID

追蹤ID （廣告名稱）提供收集連結至體驗效能之量度的機制。 在此欄位中輸入廣告名稱。

按一下右上角的&#x200B;**[!UICONTROL 下一步]**，預覽您的廣告體驗並完成啟用。

## 步驟5：預覽並啟用您的廣告

_檢閱_&#x200B;頁面會顯示您在&#x200B;_Creative設定_&#x200B;中組裝的廣告體驗，並提供檢視和編輯體驗的最後機會。 按一下&#x200B;_Creative設定_&#x200B;標籤旁的&#x200B;**[!UICONTROL 編輯區段]**&#x200B;以進行變更。 您也可以按一下右上角的「**[!UICONTROL 上一步]**」，返回&#x200B;_Creative設定_&#x200B;頁面。

### 完成廣告體驗的啟用

點擊&#x200B;**[!UICONTROL 發佈]**。完整的中繼廣告體驗及其相關中繼資料會直接推送至選取的中繼廣告管理員廣告集。 體驗會傳遞到已停用的中繼廣告管理員。 從中繼廣告管理員，您可以管理部署廣告體驗和中繼行銷活動的最後步驟。

### 登入中繼廣告管理員以完成啟用

啟用完成後，您必須登入中繼廣告管理員。 您可以從[中繼廣告管理員](https://adsmanager.facebook.com/)檢閱您的廣告體驗，並完成特定中繼頻道的發佈。
