# Adobe GenStudio for Performance Marketing整合

GenStudio for Performance Marketing可與其他功能強大的設計產品整合，為依賴這些產品的行銷人員提供順暢的體驗。 您將會在這裡找到有關可用整合以及如何使用的資訊。

探索GenStudio for Performance Marketing提供的外掛程式，以及其他Adobe產品和其他產品，以簡化您的設計工作流程。

注意：雖然這些外掛程式可讓創意人員製作廣告，並將其匯出為GenStudio的核准使用體驗，但GenStudio for Performance Marketing中的Adobe Express範本卻是創意人員製作廣告範本的建議方法。

## Figma外掛程式

GenStudio for Performance Marketing Figma外掛程式為Figma應用程式新增了面板，可讓使用者執行GenStudio for Performance Marketing動作，例如產生品牌內內容。

在[Figma外掛程式](figma-plugin.md)頁面中進一步瞭解Figma外掛程式。

## Photoshop外掛程式

GenStudio for Performance Marketing Photoshop外掛程式在Adobe Photoshop中新增了面板，可讓使用者執行GenStudio for Performance Marketing動作，例如產生品牌內內容。

在[Photoshop外掛程式](photoshop-plugin.md)頁面中進一步瞭解Photoshop外掛程式。
