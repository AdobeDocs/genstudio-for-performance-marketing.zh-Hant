# 建立連線電視體驗

在[!DNL GenStudio for Performance Marketing]中使用[[!DNL Create]](/help/user-guide/create/overview.md)在一個位置製作連線電視(CTV)廣告 — 從簡短的和共用的准則，到產生、基於場景的細分、核准和可供發佈者使用的匯出。 以下工作流程完全在[!DNL GenStudio for Performance Marketing]中執行；沒有單獨的CTV應用程式或發行者內嵌。

## 先決條件

在建立CTV廣告之前，請確認下列事項：

* 存取[!DNL GenStudio for Performance Marketing]。
* **[!DNL Brands]**、**[!DNL Products]**&#x200B;和&#x200B;**[!DNL Personas]**&#x200B;已設定為[!DNL GenStudio for Performance Marketing]中的共用物件。 請參閱[准則總覽](/help/user-guide/guidelines/overview.md)，瞭解這些物件如何通知產生。
* 建議使用Campaign資產（影片片段、影像、標誌、音樂），但並不必要；當資產遺失或不完整時，創作AI可以填補空白。

## 建立新的CTV廣告

此工作流程中的所有事情都發生在[!DNL GenStudio for Performance Marketing]內。

![連線電視建立卡](./ctv-tile.png){width="50%"}
**瀏覽至CTV建立**：

1. 登入[!DNL GenStudio for Performance Marketing]。
1. 從首頁或建立表面，移至&#x200B;**[!UICONTROL 建立]**。
1. 使用CTV建立卡選取&#x200B;**CTV**。
1. 按一下&#x200B;**[!UICONTROL 建立CTV廣告]**。

單一、簡化的CTV建立體驗隨即開啟。 您不需要先選擇廣告型別。

## 設定簡報

簡介和輸入會驅動廣告的產生方式。 您可以藉此機會為廣告產生程式提供內容與限制。

![設定簡報](./ctv-brief.png){width=80%&quot; align=&quot;center&quot;}

**設定簡報**：

1. 從您現有的共用物件中選取&#x200B;**[!DNL Brands]**、**[!DNL Products]**&#x200B;和&#x200B;**[!DNL Personas]**。
1. 直接輸入或上傳&#x200B;**創意簡介**&#x200B;以新增它。 包含行銷活動目標、關鍵訊息及任何限制。
1. 將&#x200B;**廣告持續時間**&#x200B;設為15秒或30秒。
1. 選擇性地新增&#x200B;**資產**。 上傳視訊片段、影像、標誌、音樂、畫外音或內嵌/外掛卡（拖放或檔案選擇），或從[!DNL Content]存放庫選擇資產。
1. 按一下&#x200B;**[!UICONTROL 產生]**&#x200B;按鈕。

如果資產遺失或不完整，[!DNL GenStudio for Performance Marketing]可以使用AI產生遺失的場景、音樂或旁白。 您提供的Assets一律優先於產生的素材。

[!DNL GenStudio for Performance Marketing]自動：

* 解譯摘要以及來自&#x200B;**[!DNL Brands]**、**[!DNL Products]**&#x200B;和&#x200B;**[!DNL Personas]**&#x200B;的內容。
* 組合完整的CTV廣告結構。
* 視需要建立場景、文字重疊、音樂和配音。
* 套用CTV相容的持續時間與格式。

結果會呈現完整格式的、可預覽的CTV廣告，而不是簡單的草稿時間表。

## 編輯和調整廣告

使用場景型編輯器來調整廣告，而不重新產生所有內容。

按一下場景片段中的場景，開啟它以進行編輯。 您可以執行的編輯包括：

* 用AI取代或再生單一場景。
* 編輯場景提示以建立變體。
* 重新排序或修剪場景。
* 編輯文字覆蓋圖。
* 交換、靜音或取代音樂和畫外音。
* 調整場景之間的切換。

編輯範圍已設定好，因此您可以一次重新產生一個場景，以加快版序和創意的重新整理。

>[!NOTE]
>
>編輯器不支援變更視訊片段內的物件&#x200B;** （例如，移除專案、變更產品顏色或變更人員的顯示方式）。

## 檢閱和核准

使用整合式核准工作流程提交廣告以供品牌檢閱。 品牌和利害關係人稽核者會檢查訊息、視覺效果和品牌法規遵循。 核准者會驗證廣告；他們不應代替行銷人員執行視訊編輯。

## 匯出

核准後，您可以：

* 以符合發行者要求的格式匯出最終確定的CTV廣告。
* 將廣告儲存回[!DNL Content]。
* 在下游CTV購買和販運工作流程中加以使用。

Creative旨在準備好啟動，無需重新編碼或重做。
